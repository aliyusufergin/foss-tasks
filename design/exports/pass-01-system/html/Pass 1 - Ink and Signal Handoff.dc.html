Layout notes (RN / flex intent)
Task rowRow: flexDirection row, alignItems center, bg color.bg.surface, border 1 color.border.default, radius.md, overflow hidden. Priority accent: width 3, alignSelf stretch, color.priority.* — omit entirely for "none". Content: flexDirection row, gap space.md, padding 10 14 10 11. Checkbox: 22×22, border 2 color.text.muted, radius 6; done → fill color.status.done, check icon color.accent.on, no border. Text column: flex 1, minWidth 0, gap 4. Title: type.body @ weight 500, color.text.primary, numberOfLines 1; done → color.text.muted + strikethrough. Meta line (render only when there is a signal): flexDirection row, gap space.sm, alignItems center — date chip, 13px recurrence glyph (color.text.muted), sub-task count (type.caption, color.text.muted). Pressed → row bg color.bg.pressed. Min row height 48 (hit target).
Date / deadline chiptype.caption, padding 2–3 vertical / 7–9 horizontal, radius.sm. Neutral (schedule): text color.text.secondary on color.bg.subtle. Deadline: color.accent.pressed on color.accent.surface (dark: accent.default on accent.surface). Overdue: color.status.overdue on color.status.overdueSurface. Row shows only the single most relevant chip.
Checklist item (nestable)Container: color.bg.surface, border color.border.default, radius.md, padding 10 12, column gap space.sm. Item: flexDirection row, alignItems center, gap 10 — drag handle 12px (color.border.default), checkbox 18×18 (radius 5), title type.body. Nesting: paddingLeft 26 per level. Done: same treatment as task row.
Bottom tab bar + FABBar: flexDirection row, bg color.bg.surface, borderTop 1 color.border.default, padding 8 top + safe-area bottom. Tab: flex 1, column, alignItems center, gap 3 — icon 22 stroke 2, label 10/600 (caption −1). Active color.accent.default; inactive color.text.muted. FAB: 52×52, radius.xl, bg color.accent.default, icon color.accent.on, floats space.lg from right edge above the bar; pressed → color.accent.pressed. In the mock the bar is shown as a card; in-app it is full-bleed with top hairline only.
AI-capture inputRow: flexDirection row, alignItems center, gap 10, bg color.bg.surface, border 1.5 color.accent.default, radius.lg, padding 9 vertical / 14 left / 9 right. Sparkle glyph color.accent.default. Placeholder type.body color.text.muted. Send: 32×32, radius 8, bg color.accent.default, arrow color.accent.on. AI-off variant: border color.border.default, no sparkle, plain placeholder "New task…".
Global treatmentsOverdue emphasis is always text-on-tint (never a filled red row). Done = muted + strikethrough + green check; completed rows sink to list bottom or the Done filter. Empty states: centered column, icon 40 muted, type.title line + type.body color.text.secondary hint. Offline indicator: small cloud-off glyph, color.text.muted, in the header row right side (spec'd in Pass 2).
